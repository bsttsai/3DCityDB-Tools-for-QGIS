3DCityDB-Loader plugin for QGIS



This plugin connects to the 3D City Database (v.4.x) for PostgreSQL/PostGIS and loads data as "classical" layers into QGIS. All CityGML LoDs are supported, as well as multiple citydb schemas and multiple database users. Feature attributes can be edited and changes are stored back to the database.


A quick user guide is available in the \user_guide subfolder of your plugin installation directory. Alternatively, you can download the PDF directly from here.


This plugin started as a Geomatics MSc thesis carried out at the Delft University of Technology. Since version 0.5 it is now further developed in an open collaborative approach. Check the development on GitHub.
